LIBRARY SYSTEM (WINFORMS C#)

ОПИСАНИЕ ПРОЕКТА:
Это Windows Forms приложение на языке C#.
Система библиотеки для управления книгами и журналами.

ФУНКЦИОНАЛ:
- Добавление книги
- Добавление журнала
- Просмотр списка объектов
- Редактирование элементов
- Удаление элементов
- Просмотр статистики

АРХИТЕКТУРА ПРОЕКТА:

MODELS:
- LibraryItem (базовый класс)
- Book (книга)
- Magazine (журнал)

REPOSITORY:
- Хранение данных в памяти (List)
- Add (добавить элемент)
- Remove (удалить элемент)
- GetAll (получить список)

SERVICE:
- Бизнес-логика приложения
- AddBook
- AddMagazine
- Update
- Remove
- GetAll
- GetStats

USER INTERFACE (FORM1):
- Поля ввода:
  * Название
  * Год
  * Автор / Выпуск
- Кнопки:
  * Добавить
  * Редактировать
  * Удалить
  * Статистика
- Таблица отображения данных (DataGridView)

МЕНЮ РАБОТЫ:
1. Ввод данных (название, год, автор/выпуск)
2. Добавление записи
3. Редактирование выбранной записи
4. Удаление записи
5. Просмотр статистики

ОТОБРАЖЕНИЕ ДАННЫХ:
Таблица содержит:
ID | Тип | Название | Год | Автор/Выпуск

ТЕХНОЛОГИИ:
C#
Windows Forms
ООП (наследование, полиморфизм)
Repository pattern
Service layer

ЦЕЛЬ ПРОЕКТА:
Учебный проект для изучения архитектуры приложений и работы с WinForms.
